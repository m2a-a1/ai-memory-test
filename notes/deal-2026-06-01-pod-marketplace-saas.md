## Résumé exécutif

Plateforme POD marketplace MERN stack basée en Inde, 1 an d'existence. 7,862 "subscribers actifs" déclarés mais revenus annuels seulement $11,425 → $1.45/subscriber/an = incohérence majeure. Localisation Inde, revenus non vérifiables sans domaine identifié, données contradictoires dans le listing. Asking $12,000 (réduit de $15,000). **Verdict : ❌ PASSER.**

## Audit financier

Pas de tableau mensuel disponible dans le clip.

- **Profit moyen déclaré** : $410/mo → $4,925/an
- **Multiple demandé** : $12,000 / $4,925 = **2.44x** (listing key metrics) vs **3.7x** (listing description) → **contradiction interne** 🔴
- **Spike détecté** : non évaluable sans tableau mensuel
- **Trend** : non évaluable
- **Score fiabilité** : Faible

Analyse critique des métriques déclarées :
- 7,862 "active subscribers" × 2% churn mensuel = ~157 churns/mois → implique ~157 nouvelles inscriptions/mois pour maintenir la base. À $952/mo de revenu total, revenu moyen par subscriber = **$0.12/mois**. Incohérent avec un modèle SaaS à commission ou abonnement. Ces "subscribers" sont vraisemblablement des utilisateurs freemium gratuits, non des clients payants. La métrique est présentée de façon trompeuse.
- Ecart multiple dans le listing (2.4x vs 3.7x) : une valeur est calculée sur profit ($4,925), l'autre semble incorrectement calculée. Signal de soin insuffisant dans la présentation des données.

## Red flags

🔴 **CRITIQUE — Trafic géo suspect** : vendeur localisé en Inde, marché primaire = Inde. Le vendeur indique lui-même "we will be helping you to get any support for Indian market" — suggère que la valeur du business est fortement limitée au marché indien. Transfert à un acquéreur non-indien = perte de l'avantage concurrentiel et du réseau fournisseurs.

🔴 **CRITIQUE — Revenus non vérifiables** : pas de domaine identifié pour audit Ahrefs indépendant, pas de plateforme tierce vérifiable. Screenshots uniquement.

🟡 **ATTENTION — Données contradictoires** : multiple annoncé 3.7x (description) vs 2.44x (key metrics). Signal de manque de rigueur ou de manipulation des métriques.

🟡 **ATTENTION — Metrics subscribers trompeurs** : 7,862 "active subscribers" incompatible avec $11,425/an de revenus. Ces utilisateurs sont probablement sur un tier gratuit ; la base payante réelle est bien plus petite.

🟡 **ATTENTION — Prix réduit 20%** : réduction de $15,000 à $12,000 en cours de listing → signal que le listing ne trouve pas preneur au prix initial.

ℹ️ **INFO — Stack MERN solide** : si le code est bien structuré, valeur d'actif technologique possible, mais à évaluer séparément du "business" déclaré.
