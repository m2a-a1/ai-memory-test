## Listing Description

Hi everyone,  
  
I am looking to sell my AI monitoring system to the right buyer. You can try it yourself [here](https://sentinel-ai.dev) (if you need a demo account instead of creating a real one, type *demo@acme.ai* into the email field). If you have any questions, please don't hesitate to **contact me now**.

*Details:*  
  
Sentinel is an exclusive, production-oriented SaaS control plane for enterprises and labs for building and operating AI systems in real environments. It combines **prompt management, evaluation, observability, security testing, release gating, approvals, auditability, and enterprise access control** in a single platform designed for development, staging, and production. It supports **multi-tenant organizations, memberships, roles, projects, environments, API keys, provider credentials, webhooks, integrations, audit logs, tags, and reports**, giving a buyer a serious operational foundation.  
  
Sentinel includes a **full prompt registry** with versions**, promotion workflows, candidate and active labels, experiments, structured response schemas, metadata, and approval-based go-live flows**. On the evaluation side, it supports **datasets, JSONL import, starter templates, eval suites, eval cases, judge configuration, guided scorer authoring, per-case results, baseline comparisons, run history, pass-rate and latency summaries, token usage, cost estimation, and execution metadata**.  
  
Security is equally robust, with built-in and custom **red-team packs** covering unsafe output, prompt injection, secrets leakage, PII exposure, jailbreak attempts, tool abuse, hallucinated authority, and data exfiltration, along with persisted findings, evidence, waivers, policies, and policy violations.  
  
It was specifically built for enterprise credibility, workflows are already present, including **OIDC and SAML SSO, SCIM provisioning, group-to-role mapping, domain-based login controls, just-in-time provisioning, invites, offboarding, incident management, CSV audit export, Slack alerting, and a versioned /api/v1 JSON API with a TypeScript SDK**.  
  
The codebase is a pnpm monorepo built with Next.js 15, Node.js 22+, PostgreSQL via Prisma, ClickHouse, Redis, BullMQ, MinIO or S3-compatible storage, Docker Compose, Vitest, and Playwright, with runnable SDK examples, and seeded demo data.  
  
Sentinel has not been commercialized yet, but this is a completely project by itself right now. It is a **substantial, thoughtfully engineered enterprise AI platform**. For the right acquirer, that means the hard part has already been done: the architecture, workflows, governance model, and operational depth are already in place, creating a strong foundation to launch, use it internally to gain competitve advantage, bundle into an existing product, or bring to market through an existing customer base.  
  
Thank you for reading!
