## Listing Description

**Business Overview**

GeoLoco is a fully built, production-ready B2B SaaS platform that audits Google Business Profiles (GBP) using AI, scores them on a 100-point scale, and delivers prioritised, actionable fixes to improve local search visibility. Built for SMBs, multi-location brands, and digital marketing agencies across Europe.

- Live URL: [https://geoloco.nl](https://geoloco.nl/)
- Product type: Subscription SaaS (MRR model)
- Market: Local SEO - a global multi-billion dollar category dominated by expensive enterprise tools (BrightLocal, Whitespark, Semrush Local). GeoLoco targets the underserved EU SMB + agency segment.
- Languages supported: 7 (EN, FR, DE, ES, IT, PT, NL) - fully localised UI, emails, audits and AI output.
- Branding: Clean navy + teal identity, Inter typography, polished glass-morphism UI, custom logo, video hero, full marketing site.

**Monetisation & Pricing**

Three-tier subscription model with both Stripe and Mollie (iDEAL/SEPA) integrated for global + EU coverage:

**Free** - €0 (1 location) Lead generation/trial  
**Pro** - €29/month, SMBs, freelancers  
**Agency** - €149/month, Marketing agencies, multi-location brands  
**Custom** - Negotiated, Enterprise/white-label

Includes built-in VAT handling, invoice PDF generation, and a full in-app billing portal (no Stripe dashboard required for end users).

✨ **Core Features**

For business owners

- AI-powered GBP audit scored 0–100 across multiple weighted factors (NAP consistency, photos, reviews, categories, hours, posts, Q&A, etc.)
- Priority Fixes sorted High → Medium → Low with localised, plain-language recommendations
- Review intelligence - recurring theme extraction from real Google reviews
- Keyword tracking with automatic monthly refresh (pg\_cron)
- Google Search Console integration (server-side OAuth)
- Scheduled audits + branded weekly/monthly email digests
- PDF reports with QR codes, share links, and white-label-ready styling
- Shareable public audit links (token-based, GDPR-safe)
- Milestones & gamification (5/10/25 audit achievement cards, canvas-generated)
- AI chatbot for in-app support, fully localised

For agencies

- Bulk CSV audit of dozens of locations at once
- Team management — Viewer / Editor / Admin roles
- Report colour customisation & branded exports
- Intro letter generator for client outreach (window-envelope ready)
- Custom plan billing with granular limits

For admins (you)

- Full admin panel: users, payments, blog CMS, transactional email editor, CMS-driven locale overrides, leads, tickets, analytics, sitemap health, schema drift detection
- Impersonation system for support
- Shareable checkout links (/checkout?plan=X)
- Live analytics counters, tooltip miss tracking, top pages/clicks dashboards

**Tech Stack (Modern, Maintainable)**

- Frontend: React 18 + Vite 5 + TypeScript 5 + Tailwind CSS + shadcn/ui
- Backend: Supabase (Postgres + Auth + RLS + Edge Functions + Storage + pgmq queues + pg\_cron)
- AI: Lovable AI Gateway (Google Gemini + OpenAI GPT-5 family) — no API key lock-in
- Payments: Stripe + Mollie (dual provider, EU-optimised)
- Email: Transactional system via pgmq, GDPR-compliant footers, 7-language templates
- Integrations: Google Places API (new), Google Search Console, GA4
- Hosting: Lovable Cloud (one-click publish, auto-SSL, custom domain)
- Video: Remotion framework for marketing assets

Code quality: Strict TypeScript, RLS-protected database, security-definer auth helpers, comprehensive test suite (Vitest), recent security audit completed.
