**About the Business**

Talezzi is a turnkey multilingual children's ebook platform built for the modern global family. The core insight: none of the major kids ebook platforms (Epic!, Vooks, Storyweaver) serve European diaspora communities — Turkish families in Germany, Polish families in the UK, Russian speakers across Europe, Portuguese in France — with branded, native-language picture books. Talezzi is positioned to own that underserved niche.

The whole platform is live, deployed, and SEO-indexed. The next owner just needs to bring content energy and marketing.

**Operations**

The product is fully built and deployed. Day-to-day operations require almost zero time — the platform runs on Hetzner with Docker, Google OAuth handles user signups, the admin dashboard makes book uploads a drag-and-drop affair. A new owner spends time on content creation and marketing, not maintenance. Estimated time commitment to operate: 1-2 hours per week. To grow: as much time as the owner wants to invest in stories, illustrations, and marketing.

**Customers**

Pre-launch — no active customer base yet. Target market: parents of children ages 2-7 in 10 language communities globally, with strong potential in European diaspora populations. Acquisition channels not yet activated; SEO foundation already laid for organic discovery.

**Financials**

Pre-revenue. Monthly costs: ~$15 (Hetzner VPS $10, Google Workspace $6). Stripe integration built but not activated — buyer adds their own keys to monetize.

**Tech Stack**

- Frontend: React + Vite + TypeScript + Tailwind CSS + react-i18next
- Backend: Node.js + Express + TypeScript + SQLite (better-sqlite3)
- Auth: Google OAuth 2.0
- Payments: Stripe (integrated, not activated)
- Email: Google Workspace (info@talezzi.com)
- Deployment: Hetzner VPS, Docker Compose, nginx, SSL via certbot
- Source control: Private GitHub repository

**The Talezzi Cloudlands Universe (Original IP)**

A complete, copyrighted children's book universe is part of the sale:

- Full Universe Bible (defines the world, characters, storytelling rules, visual style guide)
- Recurring characters: Luna the rabbit, Pip the owl, Echo the fox cub, Mossy the hedgehog, Wisp the cloud creature
- 3 finished stories, each translated into all 10 languages: "Luna and the Sleepy Stars," "Pip and the Question That Wouldn't Stop," "Echo's Loud Day"
- Cozy sky-kingdom aesthetic — extendable into more books, audiobooks, YouTube content, merchandise

This IP alone took weeks of focused creative work and would cost thousands to commission from a children's book author.

**Why I'm Selling**

I built Talezzi in a focused sprint and don't have the bandwidth to create the ongoing content a children's platform needs. I'd rather hand it to someone who'll run with it than let it sit unused.

**What the Buyer Brings**

**Pricing Justification**

Asset-based valuation. A solo developer would need 4-6 months of full-time work to rebuild this from scratch. The original IP (universe bible + 3 stories × 10 languages) alone would cost thousands to commission. Priced for a fast, clean handoff to the right owner.
